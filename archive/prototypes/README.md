# Archived prototypes

These prototypes are technical evidence from architecture spikes. They are not production source, are not imported by the application, and may contain assumptions or paths that no longer match the current repository.

Do not copy a prototype wholesale into a feature. Re-establish its contract, security boundary, failure behavior, and tests in the production modules under `app/`, `packages/`, `engine/`, or `native/`.

When a prototype needs Forge source for comparison, use the optional ignored official-upstream checkout at `.reference/stable-diffusion-webui-forge`. Archived paths and reports are historical evidence, never current production source locations.
