# Engine license records

Forge is materialized as a separately versioned third-party runtime and is not MIT-licensed Aureline source. A release pipeline must place the exact Forge license, source revision, patches, dependency notices, and Corresponding Source information for each runtime version in this area or in generated release metadata.

No engine binary or model is committed here. See [`THIRD_PARTY_NOTICES.md`](../../THIRD_PARTY_NOTICES.md).
